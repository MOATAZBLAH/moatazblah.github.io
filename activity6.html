6-Solar Winds article

Reflection

- I did this activity with Motaz layyas
SolarWinds attack: Collect info, create malware, distribute, exploit, communicate, harm.
Mitigation: Limit exposure, secure code, patch, secure network.
Defense tools: OSINT, code review, scanners, traffic analysis.
Understand cycle for better cybersecurity.

Phase Description Mitigation Measures Tools/Techniques
Reconnaissance Gathering information on SolarWinds software Limit exposure of sensitive information OSINT tools, online search engines, social media monitoring tools
Weaponization Creating or modifying malicious software package Secure coding practices, code reviews Integrated development environments (IDEs), code review tools, static code analysis tools
Delivery Distributing the malicious software package Secure software distribution mechanisms Malware distribution platforms, exploit kits, compromised update channels
Exploitation Exploiting vulnerabilities in SolarWinds software Patch management, vulnerability scanning Exploit frameworks, vulnerability scanners, penetration testing tools
Command and Control Establishing communication channels with attacker's infrastructure Network security controls, traffic monitoring Traffic analysis tools, network security monitoring tools, IDS/IPS
Actions on Objectives Performing malicious activities on compromised systems Access controls, data loss prevention Data exfiltration tools, lateral movement frameworks, privilege escalation tools
