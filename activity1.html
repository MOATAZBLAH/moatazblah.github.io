   IpConfig 1.1

ipconfig is a command-line tool in Windows that displays the current configuration of the IP address,
subnet mask, and default gateway for all network adapters installed on a computer. It can also display other network-related information
such as the MAC address of the adapter, the DHCP server address, and DNS server address.

   Nslookup 1.4

Nslookup is a command-line tool in Windows and other operating systems that is used to query the Domain Name System (DNS)
to obtain domain name or IP address mapping information. With nslookup, you can retrieve information
about a specific domain name or IP address, including its DNS server, IP address(es), and other related information.
